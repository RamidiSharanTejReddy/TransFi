トランスフィ・ビズペイとは？

TransFi BizPayは、グローバルな支払いと回収のための統合されたセルフサービスプラットフォームです。グローバルに事業を展開する企業が、グローバルなコンプライアンスとエンタープライズグレードのセキュリティを確保しながら、100か国以上で数分でシームレスに資金を移動できるよう支援します。

BizPayは何に使えますか？

BizPayは、グローバルに事業を展開するビジネス向けに構築された多用途のソリューションです。クロスボーダー決済を簡素化することで、企業が最も低い処理手数料、グローバルコンプライアンス、エンタープライズグレードのセキュリティで、100か国以上で支払いと支払いを行うことができます。このプラットフォームは完全にセルフサービスで、わずか 5 分で開始できます。さらに、このプラットフォームでは、地域の好みに応えたり、顧客満足度を高めたりするiBAN/バーチャルアカウントなど、250以上の支払いオプションにアクセスできます。BizPayを使用すると、企業は複数の決済チェーンにわたって正確かつ迅速に取引を行い、最速かつ最も費用対効果の高い手段を特定し、高い成功率を保証できます。

登録事業者である必要がありますか？

はい、BizPayは登録企業のみが使用できます。まだ法人として登録されていないが、間もなく事業を開始する予定であれば、登録してテスト環境で製品を試してみることができます。また、お客様のビジネスと将来のニーズを理解した上で、取引量に制限を設けたアカウントを設定することもできます。

どうすれば始められますか？どのような書類が必要ですか?

企業は直接ログインして基本的なKYBを完了して取引を開始できます。オンボーディングの所要時間はわずか5分です。パーソナライズされた体験をご希望の場合は、sales@transfi.com までご連絡ください。BizPay API 統合を含むお問い合わせやカスタムソリューションについては、営業チームにお問い合わせください。24 時間以内に対応いたします。また、本番稼働前にテスト環境を確認して、テスト API 認証情報を使用して統合を試すこともできます。

プリファンディングとはどういう意味ですか？最初にTransFiに資金を送るということですか？

プリファンディングでは、ステーブルコインを含むデジタル資産の売買や支払いなどの取引の即時処理を可能にするために、事前にTransFiに資金を入金する必要があります。これらの資金は準備金の役割を果たし、円滑で遅延のない運用を保証します。TransFiにお金を払うのではなく、グローバルな決済インフラを通じてより円滑な取引を確保し、信頼性とユーザー体験を向上させているのです。

送金にはどのくらい時間がかかりますか？

転送は、設定に応じて、即時または一括で行うことができます。即時資産取引では、受取人にリアルタイムで金額が送信され、銀行決済サイクルに従って処理されます。一括取引の場合、ステーブルコインを含む法定通貨/硬貨は、事前に合意された期限日後に一括処理され、銀行決済サイクルに従って処理されます。

トランスフィ・ビズペイは安全ですか？

はい、BizPayはPCI DSSおよびSOC2 TYPE2に準拠しているだけでなく、ISO 27001認定も受けているため、エンタープライズグレードのセキュリティとコンプライアンスを確保すると同時に、ビジネスを簡単に拡張できます。

メーカー-チェッカーの役割を割り当てて移管をリクエストおよび承認することはできますか？

管理者は、チームメイトを招待して、直接転送や一括転送の担当者や承認者などの特定の役割を割り当てることができます。メーカーが異動を開始すると、承認者は「承認」画面に表示され、承認者が確認して承認または却下することができます。このシステムにより、決済管理における責任が分散され、説明責任が強化されます。

支払いが開始されたら、リアルタイムでステータスを追跡する方法を教えてください。

BizPayダッシュボードにアクセスしてステータスを追跡できます。ステーブルコインを含むあらゆる前払い取引や資産取引について、効率的な照合と追跡に役立つ専用のエントリが用意されています。当社の多機能ダッシュボードでは、さまざまな支払いの状況を簡単に確認できます。

アカウントの詳細が間違っている場合はどうなりますか？

誤ったアカウント情報への送金は、一度開始するとすぐに失敗します。銀行によっては、失敗の確認を受け取るまでに最大 24 時間かかることがあります。

支払い回収に関する通知を受け取ることはできますか？

Webhooksを使用すると、常に最新の状態に保ち、支払いを受け取ったときにリアルタイムで通知が届き、システムを常に同期させることができます。ウェブフックを設定すると、ステーブルコインなどの決済状況の変化などのイベントをモニタリングできます。ステーブルコインはリクエストによってトリガーされ、同期と非同期の両方で処理されます。ほとんどの更新は即時に行われますが、支払い方法によっては時間がかかることがあります。TransFiは、銀行が支払い状況を確認し次第、お客様に通知します。
